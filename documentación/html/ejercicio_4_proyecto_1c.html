◆ displayNumber()

void displayNumber ( uint32_t data,
uint8_t digits,
gpioConf_t * gpio_array,
gpioConf_t * digit_array )

Muestra un número en un display multiplexado.

Convierte el número a BCD (llenando un arreglo local), luego para cada dígito:

  • pone los 4 bits en las líneas de datos (gpio_array),
  • activa el selector del dígito correspondiente (digit_array[i]) durante un breve periodo.
Parameters
[in]dataNúmero entero a mostrar.
[in]digitsCantidad de dígitos a mostrar.
[in]gpio_arrayArreglo que mapea bits b0..b3 a GPIOs (4 elementos).
[in]digit_arrayVector de GPIOs para selección de dígitos. Arreglo que mapea dígitos físicos a GPIOs (digits elementos).

Establece el estado de 4 pines GPIO a partir de un dígito BCD (0..9).

Recorre los 4 bits (b0..b3) y pone cada GPIO correspondiente en ON/OFF.

Parameters
[in]bcd_digitDígito decimal (0..9) que se representará en BCD.
[in]gpio_arrayPuntero a un arreglo de 4 elementos gpioConf_t donde gpio_array[0] corresponde a b0, gpio_array[1] a b1, etc.
